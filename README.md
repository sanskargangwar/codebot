# 🤖 CodeBot

CodeBot is a simple AI-powered chatbot built using **HTML**, **CSS**, and **JavaScript**. It provides an interactive chat interface where users can ask questions and receive AI-generated responses using an external AI API.

---

## 🚀 Features

-  Interactive Chat Interface
-  AI-Powered Responses
-  Fast and Lightweight
-  Responsive Design
-  Modern User Interface
-  Real-Time User Input
-  Chat History Display

---

##  Future Improvements

- Dark Mode
- Voice Input
- Text-to-Speech
- Export Chat as PDF
- Emoji Support
